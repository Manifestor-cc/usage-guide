EA SPORTS FC™ 26

1. Register a new Steam account​

  1. Download UrbanVPN and select Mexico, UAE or Colombia.
note

You don't have to use UrbanVPN specifically. You can use any other system-wide VPN that allows selecting a server in Mexico, UAE or Colombia.

  1. Register a new Steam account and select Mexico, UAE or Colombia in the Country of Residence field.
important

Do not register the Steam account in the app. You have to specifically do it through this page.

  1. Once you login with your newly registered Steam account in the app, disable the VPN.
note

You can uninstall the VPN at this point, as you won't need it for the rest of this guide.

2. Download the EA SPORTS FC™ 26 SHOWCASE from Steam​

3. Launch the game once so all the necessary files are created​

important

Launch the game directly via it's .exe file, not through your Steam Library or the EA app.

4. Patch the game's files​

  1. Open the game's folder (How to open Steam game's folder?)
  2. Download and open the .zip file with the patched files
  3. Select all of the files and drag them into the game's folder
  4. If a window as shown below pops up, click Replace the file in the destination. It may appear multiple times.

  1. Delete FC26_Showcase.exe
  2. Rename FC26_Showcase fixed.exe to FC26_Showcase.exe

6. Generate the Denuvo token​

  1. Open "EA.Denuvo.Token.Dumper.exe" and press Start
note

Make sure to uncheck "Add DenuvoToken to anadius.cfg even if it exists"

  1. Copy the generated Denuvo token
  2. Open anadius.cfg in the game folder
  3. Find DenuvoToken (using CTRL + F) and replace the PASTE_A_VALID_DENUVO_TOKEN_HERE with your copied token
danger

Make sure you do not erase the quotes around the token!

Congrats!​

If you've followed all of the steps correctly, you should now be able to play EA SPORTS FC™ 26.

important

Always launch the game directly via it's .exe file, not through your Steam Library or the EA app.
