## 2. Extract the files into the game's folder

1. Open the game's folder ([How to open Steam game's folder?](/extras/opening_a_steam_games_folder))
2. Open the .zip file you've downloaded
3. Select all of the files and drag them into the game's folder
4. If a window as shown below pops up, click **Replace the file in the destination**. It may appear multiple times.

![Replace files dialog](images/c3956f7a-c018-448b-9e02-973d28ed04c0-1.png)

## Congrats!

If you've followed all of the steps correctly, you should be able to play your game.

:::important
For some games, you may need to start them directly via the .exe file, not through the Steam Library.
:::
