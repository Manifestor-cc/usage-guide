Removing a game

If you want to remove a game from your library installed with SteamTools, you'll have to use the Game Remover.

1. Download and run the program​

note

The Game Remover is open-source! Check it out on GitHub

2. Click Browse and select Steam directory​

When you click Browse, it should select it automatically, you just need to click Select Folder.

If it didn't select automatically
C:\Program Files (x86)\Steam

Copy the path above and paste it into the top bar.

(The top bar)

3. Retrieve the AppID​

  1. Search for the game you want to remove on Steam (Open Steam Store search)
  2. Click on the game in the search results
  3. Copy the AppID from the URL: https://store.steampowered.com/app/70/HalfLife/ (the underlined part)

4. Remove the game from your library​

  1. Paste the AppID in the "Enter game ID" field
  2. Click on the Delete Game button
  3. Restart Steam through SteamTools

Congrats!​

If you've followed all of the steps correctly, you should have removed the game from your Steam Library.
